# Система управления сменами ЧОП

## Установка
1. Склонировать репозиторий
2. Импортировать БД из `database/init.sql`
3. Настроить `includes/config.php`
4. Использовать тестовые логины: admin/password

## Функциональность
- Управление сменами (12/24 часа)
- Медицинские осмотры
- Система бэкапов
- Разграничение прав по ролям
