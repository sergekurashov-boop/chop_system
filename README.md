# Система управления сменами ЧОП

Система для автоматизации управления сменами частного охранного предприятия.

## Текущий статус
- ✅ Базовая аутентификация и авторизация
- ✅ Управление сменами (создание, просмотр)
- ✅ База данных с тестовыми данными
- ✅ Интерфейс в серых тонах

## Установка
1. Импортируйте дамп БД из `database/init.sql`
2. Настройте подключение в `includes/config.php`
3. Используйте тестовые логины: admin/password, senior1/password

## Структура проекта
- PHP + MySQL
- Чистый CSS (без Bootstrap)
- Модульная архитектура
